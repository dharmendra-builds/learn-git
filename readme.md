# Install npm

To install npm, first install Node.js. npm is included with Node.js.

1. Download Node.js from https://nodejs.org/.
2. Run the installer for your operating system.
3. Verify installation:
   ```bash
   node -v
   npm -v
   ```

If Node.js is already installed, update npm with:

```bash
npm install -g npm
```
